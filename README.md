# Christmas Sup

A simple, festive static site.

## Structure
- `index.html` — Landing page
- `style.css` — Styles
- `script.js` — Interactions
- `images/` — Assets

## Getting started
Open `index.html` in a browser.

## Development
No build tools required. If you want a local server:

- Python 3: `python -m http.server 5500`
- Node: `npx serve .` or `npx http-server -p 5500`
